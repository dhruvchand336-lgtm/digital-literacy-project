## Project Overview
This project is created as part of the Digital Literacy course. It covers important aspects of digital skills, online presence, communication, and cyber safety. The project is divided into five tasks, each focusing on a different area of digital literacy.



## Tasks Completed

###  Task 1 – Digital Literacy Infographic
Created an infographic using Canva explaining digital literacy, online safety, and useful tools for students.

###  Task 2 – Digital Portfolio
Created professional profiles on GitHub, LinkedIn, and Kaggle to build a strong digital presence.

###  Task 3 – Platforms Exploration
Solved a coding problem on HackerRank and created a Google Form quiz for digital literacy awareness.

###  Task 4 – Email Etiquette
Drafted professional emails and created a checklist for responsible social media usage.

###  Task 5 – Cybercrime Awareness
Prepared a case study on UPI fraud and created a prevention checklist for online safety.
